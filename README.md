# это не gpt это я на предыдущей работе привык работать с readme

Минимальное PHP-приложение, упакованное в Docker (php-fpm + nginx). Позволяет ввести год и получить рассчитанное время наступления “пилотируемого Нового года”.

## Как запустить

Из корня проекта выполните две команды:

```bash
docker build -t pilot-new-year .
docker run --rm -p 8080:80 pilot-new-year
```

## Где открыть

После старта контейнера страница доступна в браузере по адресу `http://localhost:8080/`. Введите год, нажмите “Посчитать” — результат и история запросов появятся ниже формы.
